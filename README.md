# Job Market Analytics & Automated Extraction Engine

An automated data pipeline and RESTful API designed to extract, aggregate, and analyze remote tech job market trends using **Python**, **FastAPI**, and **MySQL**.

## 🚀 Key Features

* **Automated Data Pipeline:** Extracts structured remote job posts and skill tags.
* **Database Integration:** Relational MySQL storage using SQLAlchemy ORM.
* **FastAPI Backend:** Clean endpoints with dynamic pagination and filtering.
* **Analytics Engine:** Dynamic endpoint calculating real-time top demanded skills.

## 🛠️ Tech Stack

* **Language:** Python 3.10+
* **Framework:** FastAPI / Uvicorn
* **Database:** MySQL
* **ORM:** SQLAlchemy / PyMySQL
* **Data Processing:** Requests, Collections
